R code in Markdown

You can insert plots by using including the code chunk like below:

```{r plot_sepal, fig.cap="Figure title here", fig.height=8, fig.width=8, echo=FALSE}
plot(Sepal.Length ~ Sepal.Width, data = iris)
```

Figure title here

… The options added to the code chunk tell it to add a caption, and set the height and width of the figure.

You can also create tables by using the kable() in the knitr package.

```{r}
knitr::kable(head(iris), caption = "Table caption here")
```

You can hide the code chunk but keep the output by using the echo=FALSE option:

```{r, echo=FALSE}
model <- lm(Sepal.Length ~ Sepal.Width, data = iris)
coef(model)
```

#> (Intercept) Sepal.Width 
#>   6.5262226  -0.2233611

You can use inline R code too, to insert numbers directly into the document.

Use this: The estimate is `r round(coef(model)[2], 2)`… to get this: The estimate is -0.22.

If you want to insert a citation use [@Hoejsgaard2006a] to get it to look like: (Højsgaard, Halekoh, and Yan 2006)… the reference is inserted onto the bottom of the document. You need to add a line like this bibliography: bibfile.bib to the top YAML header (e.g. right below where you put the title and author)
