Session material

This contains the material used in the Aarhus University Open Coders hands-on coding sessions. Below is the list of sessions taught to date, according to skill level.

Make sure to follow the installation instructions.

Series:

Beginner:

  • Introduction to visualising and plotting your data using ggplot2
  • Work with and manage your data efficiently in R with dplyr and tidyr
  • Making reproducible documents with R

Package Development:

  • Setup and start creating your own package in R
  • Create your own functions for package development in R

Analysis workflow:

  • Keeping track of changes to your files with version control

Misc sessions:

  • Creating your own functions in R
